# 1：模块

## 1.1：系统设置模块 setting

### 1.1.1：用户模块：登陆模块

- 涉及到数据字典模块信息的查询

## 1.2：工作台(核心业务)：workbeach

- 市场活动模块：activity
  - 点击创建按钮，打开模态窗口
  - 添加操作
  - 展现市场活动信息列表(结合条件查询加分页查询)
  - 全选/反选
  - 执行删除操作(可批量删除)
  - 点击修改按钮打开修改操作的模态窗口
  - 点击更新执行修改操作
  - 点击市场活动名称跳转到详细信息页，展现详细信息
  - 详细信息页加载完毕后，展现备注信息列表
  - 备注添加修改删除
- 线索模块：clue
  - 点击创建按钮，展现创建模块窗口
    - 重点下拉列表的加载，使用服务器缓存中的数据字典
  - 点击线索名称进入详细信息页，展示线索的详细信息
  - 在页面加载完毕后，展现关联的市场活动列表
  - 可以执行解除关联操作
  - 关联市场互动的操作
  - 点击转换按钮跳转到线索转换页面
  - 执行线索转换的操作(同时可添加交易)
- 交易模块：transaction
  - 点击创建按钮，跳转到交易添加页
  - 点击交易名称，进入交易的详细信息页
  - 展现交易历史列表
  - 动态展现交易阶段内容和图标
  - 点击阶段图标更改交易阶段

- 统计图表
  - 交易阶段的统计图表：ECharts
